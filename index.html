Introduction

I’m a UI/UX design leader with over 15 years of experience in the industry. I started my career in print and display graphics, moved on to flash animation & development which then evolved into video editing and motion graphics. This breadth of experience has enabled me to better understand and serve my users in whatever media they choose to use.

My current focus is crafting easily navigable experiences by designing simple, attractive interfaces that are accessible to various types of users.

Featured Work

TDS TV+ Product Launch

Project Details

Deliverables: Web/UI Design. HTML/CSS Development and Review. Page Flows. User Journeys. Technical Requirements.

Summary: Using concepts and some artwork provided by another team, my primary task was creating a website experience that matched the print and video collateral and would highlight the latest technological features offered by TDS’ newest TV product. In addition to that, I worked closely with the project managers and software engineers to define requirements and provide HTML/CSS for some of the features that came with the upgraded product.

Collaborators: Graphic Designers, Copywriters, Marketing Managers, Product Management, Web Analyst, Software Engineers, Project Managers.

TDS Telecom Careers Microsite

Project Details

Deliverables: Web Design. HTML/CSS Development and Review. User Testing. Content Strategy.

Summary: The tdstelecom.com/careers site design was updated to reflect a more contemporary look and feel and to showcase images of real TDS employees in order to provide a glimpse of actual experiences they’ve had with the company. The main site was also modified to make it easier for applicants to start job searching. Much of the existing content was pared down so that potential recruits are able to access the information they need as quickly as possible.

Collaborators: Web Analyst, HR Talent Acquisition Team

TDS Telecom e-Commerce Site

Project Details

Deliverables: Web/UI Design. HTML/CSS Development and Review. Page Flows. Wireframes. Interactive Prototypes. User Journeys. Technical Requirements. AEM Component Requirements. Wiki Documentation.

Summary: The tdstelecom.com redesign was an enormous undertaking that took a lot of teams and a few years before it was considered complete. Not only was the site moving to a different Content Management System (CMS), it needed to become responsive and able to adapt to any screen width without losing the integrity of the experience.

Collaborators: This list includes but is definitely not limited to Copywriters, Marketing Managers, Product Management, Web Analyst, Front End Developers, Software Engineers, and Project Managers. Departments consulted for site content include Customer Service, Repair, Legal, and Public Relations.
